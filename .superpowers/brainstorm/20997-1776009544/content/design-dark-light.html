What gets fixed

Dark mode settings — Replace boxy icon containers with inline SVG icons. iOS-style grouped list with hairline separators. No heavy borders.
Light mode contrast — All text uses var(--foreground) not hardcoded rgba. Card surfaces get opaque white backgrounds so numbers are always readable.
Consistent token usage — Audit all components for hardcoded colors. Everything must reference CSS custom properties so light/dark switching works automatically.
Full dark mode review — Every page checked for dated styling, heavy borders, low-contrast elements, or cold/blue tones that don't match the warm charcoal brand.
