Key Changes

Reduced height — Smaller padding, 9px labels, tighter icon-to-label gap. Nav shrinks ~30% vertically.
Accent-colored (+) button — Uses the user's chosen accent color with gradient + glow shadow. Elevated above the bar.
Subtler glass effect — Lower opacity backdrop, near-invisible top border. Premium, not "frosted glass demo."
Inactive items at 50% opacity — Active item full brightness with no pill background. Cleaner, Vogue-like restraint.
(+) Quick Actions — Tap opens a bottom sheet with 3 options: Add Garment, BURS Live Scan, Bulk Add. Sheet slides up with backdrop blur.
