iOS-native design decisions

SF Pro-style icons — Rounded strokes, 1.5px weight, rounded caps/joins. Filled state for active tab. Matches iOS tab bar conventions.
System font stack — Falls back to -apple-system / SF Pro Text first, DM Sans as web fallback. 10px labels with -0.1 letter-spacing like native iOS.
0.5px separator — iOS uses hairline borders (0.5px), not 1px. Subtler, more refined.
Order: Home · Wardrobe · (+) · Plan · Insights — Home first (most visited), creative actions center, utility right.
(+) as compact pill — Sits inside the bar, no elevation. Wider than tall (44×36), matching iOS action button proportions. Accent gradient.
